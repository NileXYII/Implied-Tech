Why KENAI

Everything you need.
Nothing you don't.

Built directly from the Material Design 3 specification. Every token, every component, every motion curve — exactly as Google intended.

palette
Full M3 Color System

Primary, secondary, tertiary, error, surface containers — all 30+ color roles, light and dark, with proper tonal elevation overlays.

text_fields
15-Role Typography Scale

Display, Headline, Title, Body, Label — 3 sizes each. Powered by Google Sans + Roboto, with exact M3 tracking and line heights.

animation
M3 Motion Tokens

All 16 duration steps and 9 named easing curves — including Emphasized Decelerate, Standard, and Spring — as CSS custom properties.

water_drop
Ripple + State Layers

Pixel-perfect M3 ink ripple engine via kenai.js. Every interactive element automatically gets correct hover (8%) and pressed (12%) state layers.

dark_mode
Light & Dark — Automatic

Flip a single data-theme attribute. Respects prefers-color-scheme, persists to localStorage, fires a custom event.

bolt
Zero Build Step

Pure CSS custom properties — no Sass, no PostCSS, no webpack required. Drop two files in your project and ship.

Design Tokens

Every value is a CSS variable.

Access the full M3 token system directly. Override anything with one line of CSS. No configuration files required.

Color Tokens 
--md-primary:           #1B6EF3;
--md-on-primary:        #FFFFFF;
--md-primary-container: #D8E3FF;
--md-secondary:         #565F71;
--md-surface:           #F9F9FF;
--md-surface-container: #EDEDF4;
--md-outline:           #74777F;
--md-error:             #BA1A1A;

/* Usage */
.my-button {
  background: var(--md-primary);
  color: var(--md-on-primary);
}
Motion Tokens 
--md-duration-short-4:  200ms;
--md-duration-medium-2: 300ms;
--md-duration-long-2:   500ms;

--md-easing-standard:
  cubic-bezier(0.2, 0, 0, 1);
--md-easing-emphasized-decel:
  cubic-bezier(0.05, 0.7, 0.1, 1);
--md-easing-spring:
  cubic-bezier(0.34, 1.56, 0.64, 1);

/* Usage */
.panel {
  transition: transform
    var(--md-duration-medium-2)
    var(--md-easing-emphasized-decel);
}
Dynamic Color 
// Generate full M3 palette
// from any seed hex color
KENAI.Color.applyPalette('#4CAF50');

// Reset to KENAI defaults
KENAI.Color.reset();

// Theme management
KENAI.Theme.toggle();
KENAI.Theme.apply('dark');
KENAI.Theme.current(); // → 'dark'

// Listen for changes
document.documentElement
  .addEventListener(
    'kenai:themechange',
    e => console.log(e.detail.theme)
  );
Shape + Elevation 
/* Shape scale */
--md-shape-extra-small: 4px;
--md-shape-small:       8px;
--md-shape-medium:      12px;
--md-shape-large:       16px;
--md-shape-extra-large: 28px;
--md-shape-full:        9999px;

/* Elevation */
--md-elevation-1: 
  0px 1px 2px rgba(0,0,0,.3),
  0px 1px 3px 1px rgba(0,0,0,.15);

/* Utility classes */
.rounded-lg  → border-radius: 16px
.elevation-2 → shadow + tinted bg
.p-4         → padding: 16px
.gap-3       → gap: 12px
